Justificación metodológica

El desarrollo de este proyecto se realizó utilizando una metodología estructurada, comenzando con la planificación de la interfaz.

Se utilizó Sass para organizar los estilos en módulos, permitiendo un código más ordenado, reutilizable y fácil de mantener.

La estructura del proyecto separa correctamente HTML, CSS y recursos, siguiendo buenas prácticas de desarrollo front-end.

Además, se incorporaron imágenes para mejorar la presentación visual de la aplicación.

Este enfoque permite escalar el proyecto en el futuro y facilita su mantenimiento.
