SmartBudget
Aplicación web básica que permite gestionar ingresos y gastos de manera simple.

Funcionalidades
Visualización de interfaz principal
Diseño responsive básico
Sección de control financiero
Uso de Sass para organización de estilos

Tecnologías utilizadas
HTML5
CSS3
Sass (SCSS)

Estructura del proyecto
/css → estilos compilados
/sass → archivos SCSS organizados
/img → imágenes utilizadas
index.html → página principal

Objetivo

Aplicar buenas prácticas de desarrollo front-end utilizando Sass y estructura modular.
